Project HawkEye is a FinTech Project which starts from Scratch of Programming to building a Complete Wealth Management Portal. The Project started with 100+ Crew Members, with 15-20 Core Members to work on building the Actual App and Web Platform with Integrated Chatbots using Data-driven Approach. The Portal will run on advanced Financial Algorithms, which will assist in Intelligent Decision Making. The project was divided into two Teams:- Finance and Technology. Finance team worked on Financial Data and Making Algorithms and Technology team worked on making Interface
